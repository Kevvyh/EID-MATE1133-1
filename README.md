# Analizador y Visualizador de Límites

Este proyecto corresponde a la Evaluación Integrada de Desempeño del curso MATE1133 - Cálculo Básico de la Universidad Católica de Temuco. Fue desarrollado con el objetivo de liderar la creación de una plataforma tecnológica de apoyo académico.

## 🎯 Características Principales

* **Evaluación de Límites:** Permite al usuario ingresar una función matemática *f(x)* y evaluar su límite cuando la variable tiende a un valor *h*. Soporta límites laterales y límites al infinito.
* **Representación Visual:** Grafica la función ingresada para ofrecer una visualización clara del comportamiento de la curva al acercarse al punto evaluado.
* **Interfaz de Usuario (GUI):** Cuenta con un diseño moderno, limpio y amigable para facilitar la interacción del usuario.

## 🛠️ Tecnologías y Librerías

El proyecto está desarrollado íntegramente en Python e implementa el uso de las siguientes librerías obligatorias:

* **SymPy:** Herramienta fundamental utilizada para el manejo de variables simbólicas y la evaluación analítica de expresiones matemáticas.
* **CustomTkinter:** Librería potente utilizada para la creación de una interfaz gráfica de usuario (GUI) moderna.
* **Matplotlib:** Librería esencial para la generación y renderizado de las gráficas dentro de la aplicación.
